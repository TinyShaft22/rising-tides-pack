# Connectors

This plugin works standalone. Connectors enhance it.

## Required
None

## Recommended
None

## Optional
- **context7** — Live documentation for React and UI libraries.
- **shadcn** — Component registry for browsing and installing shadcn/ui components directly.

## Setup
```bash
claude mcp add context7 --scope project -- npx -y @upstash/context7-mcp
claude mcp add shadcn --scope project -- npx -y @jpisnice/shadcn-ui-mcp-server
```
